# 操作手册

## 1. 启动前准备
1. 打开项目根目录：
   `F:\JavaWork\management-system\management-system`
2. 确认 Java 可用：
   `java -version`
3. 确认 Node.js 可用：
   `node -v`

## 2. 后端测试
1. 在项目根目录打开终端
2. 运行：

```bat
mvnw.cmd test
```

3. 预期结果：
   测试全部通过

## 3. 前端依赖安装
1. 进入前端目录：

```bat
cd frontend
```

2. 安装依赖：

```bat
npm install
```

## 4. 前端构建验证
在 `frontend` 目录运行：

```bat
npm run build
```

预期结果：
- 构建成功
- 允许出现大体积 chunk 警告

## 5. 启动系统

### 方案一：一键启动（推荐）

在项目根目录运行：

```bat
start_all.bat
```

脚本会自动：
- 检查 Java / Node 环境
- 启动后端（8080）
- 等待后端健康检查
- 启动前端（5173）
- 自动打开浏览器

### 方案二：手动启动（备用）

后端：

```bat
cd /d F:\JavaWork\management-system\management-system
mvnw.cmd spring-boot:run
```

前端：

```bat
cd /d F:\JavaWork\management-system\management-system\frontend
npm run dev
```

## 6. 访问地址
- 前端：`http://localhost:5173`
- 后端：`http://localhost:8080`
- Swagger：`http://localhost:8080/swagger-ui.html`

## 7. 登录账号
- 管理员：`admin / 123456`
- 人事：`hr / 123456`
- 员工：`employee / 123456`

## 8. 最终验收步骤
1. 使用 `admin` 登录，检查仪表盘、用户、角色、公告、培训、项目管理
2. 使用 `employee` 登录，检查个人中心、考勤打卡、请假申请、我的异动
3. 使用 `hr` 或 `admin` 登录，检查请假审批、异动审批、生效执行
4. 检查“转”相关功能：
   - 岗位调动
   - 职级调整
   - 异动记录展示
5. 再执行一次：

```bat
mvnw.cmd test
```

6. 如需验证前端构建：

```bat
cd frontend
npm run build
```

## 9. 常见问题

### 9.1 前端启动失败
- 先执行 `npm install`
- 再执行 `npm run dev`

### 9.2 后端启动失败
- 检查 Java 版本是否为 17+
- 检查 8080 端口是否被占用

### 9.3 页面打不开
- 确认后端和前端终端都没有报错
- 确认访问的是 `http://localhost:5173`

### 9.4 切换 MySQL
1. 创建数据库 `management_system`
2. 导入 `navicat_import.sql` 或 `src/main/resources/schema.sql`
3. 配置环境变量：`DB_URL`、`DB_USERNAME`、`DB_PASSWORD`
4. 运行：

```bat
start_all.bat
```
